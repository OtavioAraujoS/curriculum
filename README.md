# 🎨 Otávio Araujo - Modern Portfolio

Um portfólio moderno, responsivo e totalmente bilíngue (Português/Inglês) desenvolvido com React/HTML/CSS/JAVASCRIPT.

## ✨ Características

### Design & Experiência

- **Design Editorial Premium**: Inspirado no aesthetic minimalista e sofisticado
- **Layout Assimétrico**: Composição visual única e moderna
- **Animações Suaves**: Transições elegantes e micro-interações
- **Totalmente Responsivo**: Funciona perfeitamente em desktop, tablet e mobile
- **Bilíngue**: Suporte completo para Português (Brasil) e Inglês

### Seções Incluídas

1. **Hero Section**
   - Apresentação impactante com nome e título

2. **Companies Section**
   - Display das empresas onde trabalhei

3. **About Section**
   - Descrição profissional
   - Métricas destacadas

4. **Specialties**
   - Descrições detalhadas das especialidades

5. **Experience Timeline**
   - Timeline visual co posições profissionais
   - Períodos, cargo, empresas e descrições

6. **Tech Stack**
   - Categorias: Frontend, Backend, Database, Tools
   - Badges de tecnologias

7. **GitHub Projects**
   - Integração automática com GitHub API
   - Carrega últimos projetos
   - Exibe estrelas, linguagem e descrição

8. **Contact Section**
   - Cards de contato
   - Links diretos
   - Design moderno e interativo

### Funcionalidades Técnicas

✅ Language Toggle (PT-BR / EN)<br>

## 🚀 Como Usar

### Instalação

1. Clone o repositório:

   ```bash
   git clone https://github.com/OtavioAraujoS/curriculum.git
   ```

2. Instale as dependências:

   ```bash
   pnpm install
   ```

3. Execute o servidor de desenvolvimento:

   ```bash
   pnpm dev
   ```

4. Abra [http://localhost:5173/](http://localhost:5173/) para acessar o dashboard.
